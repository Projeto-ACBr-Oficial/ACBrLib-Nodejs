Class ACBrLibMDFeMT

Description

ACBrLibMDFeMT é uma classe de alto nível que abstrai os métodos da ACBrLibMDFe Multi-thread
Esta classe permite que programadores de javascript/typescript usem a ACBrLibMDFe sem grandes preocupações.
